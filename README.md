# cpp-library-management-system
A console-based Library Management System built using C++ that allows users to add, search, issue, return, and delete books using file handling.

A console-based Library Management System developed using **C++**, demonstrating Object-Oriented Programming (OOP) concepts and binary file handling for persistent data storage.

---

## 🚀 Project Overview

This project allows users to manage books in a library system through a menu-driven console interface. All records are stored in a binary file to ensure data persistence between program executions.

It is a beginner-friendly yet resume-worthy C++ project.

---

## ✨ Features

* ➕ Add new books
* 📖 View all books
* 🔍 Search book by ID
* 📕 Issue book
* 📗 Return book
* ❌ Delete book
* 💾 Persistent storage using binary files

---

## 🛠️ Technologies Used

* C++
* Object-Oriented Programming (OOP)
* File Handling (`fstream`)
* Binary File Storage
* Menu-driven Programming
